# FluentWeb.js

Bringing the **Windows 11** experience to the web.
(Japanese日本語訳)
(Webブラウザで、Windows 11の体験を。)
## Features / 機能
* **Windows 11-style Mica effects**: Native-like translucent blur.
(Windows 11風マイカ効果: ネイティブアプリのような半透明なぼかしを実現しております。)
* **Modern Title Bars**: Fully functional and draggable.
(モダンなタイトルバー:快適なドラッグ移動とOSライクな挙動を実装しております。)
* **Open Source**: Released as **OSS**.
(オープンソース: OSSとして公開しております。)
